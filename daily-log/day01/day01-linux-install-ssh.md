# Day 1 - Linux 서버 첫 설치 및 SSH 접속

## 1. 설치 환경
- OS: Ubuntu Server 22.04 LTS
- 설치 방식: GUI 미설치 (CLI 환경)
- 가상화 환경: VirtualBox
- 사용자 계정: ops
- OpenSSH Server 설치

## 2. 서버 로그인 및 기본 확인
- 로컬 콘솔에서 서버 로그인 성공
- pwd 명령어로 홈 디렉토리(/home/ops) 확인
- ls, mkdir, cd 등 기본 명령어 사용

## 3. 네트워크 상태 확인
- ip a 명령어로 네트워크 인터페이스 확인
- enp0s3 인터페이스에 IP 주소(10.0.2.15) 정상 할당
- 인터페이스 상태 UP 확인

## 4. SSH 원격 접속
- SSH 최초 접속 시 호스트 키 검증 메시지 확인
- 서버 호스트 키 신뢰 등록 후 접속
- 원격 SSH 접속 성공

## 5. 정리
- 서버는 GUI 없이 CLI 기반으로 운영됨
- 네트워크 및 SSH 접속이 서버 운영의 기본임을 확인
